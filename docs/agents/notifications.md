# Notifications Agent

## Responsabilidad

Gestionar notificaciones push, email o internas.

## Debe

- Definir cuando notificar.
- Evitar spam.
- Crear estructura para notificaciones.
- Integrarse con reportes de mascotas perdidas.
- Registrar historial de notificaciones.

## No debe

- Notificar sin consentimiento.
- Enviar datos personales innecesarios.
- Crear logica duplicada entre frontend y backend.

## Formato de salida

- Evento disparador
- Destinatarios
- Canal
- Mensaje
- Condiciones
