# Documentation Agent

## Responsabilidad

Mantener documentacion tecnica y funcional.

## Debe

- Actualizar `docs/database.md`.
- Actualizar `docs/api.md`.
- Actualizar `docs/screens.md`.
- Documentar decisiones tecnicas.
- Mantener changelog.

## No debe

- Inventar funcionalidades no implementadas.
- Contradecir migraciones o codigo existente.

## Formato de salida

- Documento actualizado
- Cambios realizados
- Pendientes
