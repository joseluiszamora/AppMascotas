# Components

## Componentes base

### 1. Bottom navigation

#### Estilo

- Negra.
- Flotante.
- Super redondeada.
- Compacta.
- Premium.

```css
background: #111111;
border-radius: 999px;
height: 72px;
```

#### Tab activo

```css
background: #E89B1C;
```

#### Iconos

Usar:

- Outline icons.
- Minimalistas.
- White icons.

### 2. Hero banner

#### Caracteristicas

- Gradiente naranja.
- Mascota grande.
- CTA visible.
- Bordes suaves.

#### Estilo

```css
height: 180px;
border-radius: 28px;
```

### 3. Search bar

#### Diseno

```css
background: white;
height: 52px;
border-radius: 999px;
```

#### Debe incluir

- Icono busqueda.
- Placeholder suave.
- Boton filtros circular naranja.

### 4. Cards

#### Diseno general

Todas las tarjetas deben:

- Flotar visualmente.
- Tener mucho padding.
- Usar colores pastel.
- Tener sombras suaves.
- Bordes grandes.

#### Estilo base

```css
border-radius: 28px;
padding: 18px;
background: white;
```

### 5. Cards de mascotas

Deben incluir:

- Imagen grande.
- Nombre.
- Tipo.
- Estado.
- Informacion rapida.

#### Fondo

Usar fondos pastel:

- Rosado.
- Verde.
- Azul.
- Amarillo.
- Morado.

### 6. Inputs

#### Estilo

```css
background: #F2F2F2;
height: 56px;
border-radius: 18px;
```

#### UX

- Sin bordes negros.
- Sin outline duro.
- Focus suave.
