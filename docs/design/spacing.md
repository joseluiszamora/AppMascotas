# Spacing

## Sombras

Las sombras deben ser muy suaves.

```css
box-shadow: 0 8px 24px rgba(0, 0, 0, 0.06);
```

Nunca usar:

- Sombras oscuras.
- Sombras duras.
- Glow exagerado.

## Layout

### Safe areas

Disenar pensando en:

- iPhone moderno.
- Dynamic Island.
- Bordes curvos.

### Espaciado

Padding general:

```css
padding: 20px;
```

Separacion vertical:

```css
margin-bottom: 24px;
```

Gap interno:

```css
gap: 12px;
```

## Diseno mobile first

Disenar siempre primero para:

```text
390 x 844
```

Referencia: iPhone moderno.
