# Animations

## Principios

Las animaciones deben sentirse:

- Suaves.
- Naturales.
- Premium.

## Recomendadas

- Fade.
- Slide-up.
- Floating transitions.
- Scale subtle.
- Spring animations.

## Duracion

```css
transition: 0.25s ease;
```

## Motion guidelines

Evitar:

- Rebotes exagerados.
- Animaciones lentas.
- Transiciones bruscas.
