# Accessibility

## Accesibilidad

Usar:

- Contraste suave pero legible.
- Tap targets grandes.
- Tipografia clara.
- Espaciado amplio.

## Reglas relacionadas

- Priorizar simplicidad y uso mobile.
- Evitar botones pequenos.
- Evitar layouts densos.
- Mantener informacion importante visible y facil de encontrar.
- Asegurar que el usuario pueda navegar con una sola mano.
