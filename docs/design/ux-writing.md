# UX Writing

## Estilo de texto

Usar tono:

- Emocional.
- Cercano.
- Positivo.
- Comunitario.

## Ejemplos

- Ayuda a que vuelva a casa.
- Mascotas cerca de ti.
- Comparte el reporte rapidamente.
- Tu ayuda puede marcar la diferencia.

## Reglas UX criticas

El usuario debe poder:

- Reportar una mascota en menos de 2 minutos.
- Encontrar informacion rapidamente.
- Navegar intuitivamente.
- Usar la app con una sola mano.

## Errores comunes a evitar

No usar:

- Bordes cuadrados.
- Sombras agresivas.
- Gradientes fuertes.
- Texto excesivo.
- Interfaces recargadas.
- Colores saturados.
- Botones pequenos.
- Cards apretadas.
- Layouts densos.
