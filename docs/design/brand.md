# Brand

## Objetivo general

Disenar una aplicacion movil de mascotas con estetica:

- Premium.
- Moderna.
- Minimalista.
- Emocional.
- Comunitaria.
- Tipo startup tecnologica.
- Calidad Dribbble / Behance.
- Inspirada en apps modernas iOS.

La interfaz debe sentirse:

- Suave.
- Elegante.
- Amigable.
- Muy visual.
- Ligera.
- Moderna.
- Calida.
- "Instagram + Airbnb + Uber para mascotas".

## Concepto visual principal

### Sensacion de marca

La app debe transmitir:

- Amor por mascotas.
- Seguridad.
- Confianza.
- Comunidad.
- Cuidado premium.
- Tecnologia moderna.
- Diseno emocional.

## Principios de diseno

### 1. Mucho espacio en blanco

La UI nunca debe verse saturada.

Usar:

- Margenes amplios.
- Separaciones generosas.
- Aire visual.

### 2. Bordes exageradamente redondeados

Todo debe verse suave y amigable.

Ejemplos:

```css
cards: 28px;
buttons: 999px;
inputs: 18px;
images: 24px;
bottom-nav: 999px;
```

### 3. Fotografias grandes

Las mascotas son protagonistas.

Usar:

- Hero images grandes.
- Fotos emocionales.
- Recortes limpios.
- Fondos suaves.

Nunca usar imagenes pequenas o comprimidas.

### 4. Colores pastel premium

Evitar colores agresivos.

Usar:

- Pasteles calidos.
- Tonos suaves.
- Contrastes elegantes.

### 5. UI minimalista

Eliminar:

- Bordes innecesarios.
- Lineas fuertes.
- Decoraciones excesivas.
- Texto largo.

## Decoracion visual

### Fondos

Usar:

- Paw prints.
- Blob shapes.
- Gradientes suaves.
- Circulos organicos.

### Opacidad

```css
opacity: 0.08;
```

Nunca competir con contenido principal.

### Ilustraciones

Usar:

- Mascotas tiernas.
- Expresivas.
- Realistas.
- Fondos transparentes.

### Estilo fotografico

Las mascotas deben verse:

- Alegres.
- Adorables.
- Premium.
- Bien iluminadas.

## Recomendacion final

La esencia visual del producto debe sentirse como:

- Producto startup premium.
- App emocional.
- Diseno suave y moderno.
- Interfaz adorable.
- Mucho aire visual.
- Fotos grandes.
- Componentes flotantes.
- Pasteles suaves.
- UX extremadamente simple.

El usuario debe abrir la app y sentir:

> Esta app ama a las mascotas.
